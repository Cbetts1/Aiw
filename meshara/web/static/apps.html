AIM Browser

The AIM Browser Experience

AIM is not just a website — it is a browser-native mesh experience. Every tab is a window into a live AI node.

🔒 aim.visionfortomorrow  /  AIM Artificial Intelligence Mesh

Your browser is your AIM portal

Open http://localhost:8080 after starting the AIM web bridge and your browser becomes a full AIM interface — no app installation, no plugins, no extensions needed.

The AIM web bridge (aim web start) runs a lightweight HTTP server that forwards browser requests to your local AIM node. Accessible on any device connected to your network — desktop, tablet, or phone.

To make AIM publicly accessible via a custom domain (e.g. aim.visionfortomorrow): point your domain's DNS A-record to your server, then run aim web start --port 80 or use nginx as a reverse proxy.
