Publish to the World Wide Web

Register your site on the AIM mesh and bridge it to the traditional web — fully self-contained, no external services required.

Register Your Site

Generate a Full Static Snapshot

aim www publish --out ./my-www-site --aim-url http://localhost:8080

Run this command to export all AIM pages and live content into a plain HTML folder you can deploy to GitHub Pages, nginx, or any static host.


aim www start --port 80

Start the AIM web bridge on standard HTTP port 80 (may require root/sudo on some systems).
